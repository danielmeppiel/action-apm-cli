# test-project

An APM (Agent Primitives Manager) application.

## Quick Start

```bash
# Install dependencies
apm install

# Run the hello world prompt
apm run --param name="<YourGitHubHandle>"

# Preview before execution
apm preview --param name="<YourGitHubHandle>"
```
